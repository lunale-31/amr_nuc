# amr_nuc
Mobile robot powered through ROS2 and NUC board. 
Components used: 
    1) RMCS Motor drivers and DC servo motor with encoders 
    2) INTEL NUC board
    3) Rplidar A1
    4) Buck Converter
    5) 12v Lead Acid Quanta Batteries (x2)
    6) 2 Standard Wheels 
    7) 4 Castor Wheels
    8) USB to TTL Converter
    9) Jumper Wires
    10) Emergency Stop Button
